# damage-detective

Your instructions are to create an application that can handle multiple images being uploaded of property damage claims and analyze through the GPT4 API the damage that can be detected alongside the room classifications in one full analysis displayed once upload is complete

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/damage-detective.git
cd damage-detective
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
